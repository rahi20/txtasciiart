## What does txtaasciiart do ?
`txtasciiart` prints out ASCII Art of the arguments passed to it.

Example : 
```bash 
txtasciiart hello world
```
prints out : 

```
# # ### #   #    #      # #  #  ##  #   ##  
# # #   #   #   # #     # # # # # # #   # # 
### ##  #   #   # #     ### # # ##  #   # # 
# # #   #   #   # #     ### # # # # #   # # 
# # ### ### ###  #      # #  #  # # ### ##  
```
